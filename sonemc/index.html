SoneMC Minecraft Plugins

Customizable Minecraft plugins for Spigot/Bukkit/Paper built for versions 1.20.2+

About

SoneMC is a personal project where I create and share simple Minecraft plugins for Spigot, Bukkit, and Paper servers. While these plugins might not be top-tier in terms of features or performance, they are functional and designed to be easy to use. My goal is to provide useful tools for Minecraft server owners who need lightweight solutions.

All the plugins are developed and maintained by me, and I strive to make them as user-friendly as possible. Thank you for support!

Features

Easy to Use

Our plugins are designed to be as user-friendly as possible. Many functions can be easily customized. You can also see Documentation, if you dont know what to do.

Regular Updates

We continuously update our plugins to ensure compatibility with newer Minecraft versions, including 1.21 and beyond. Bugs and issues are fixed as soon as they are discovered.

Frequently Asked Questions

How do I install the plugins?

Simply download the plugin, place it in the plugins folder of your server, and restart your server. Configuration is optional but recommended.

Are the plugins compatible with all versions?

The plugins are built for Minecraft version 1.20.2+ but may work on older versions.

How do I report a bug?

You can report bugs through our GitHub issues page or contact us directly via Discord. We appreciate your feedback!

Can I use multiple plugins at the same time?

Absolutely! Our plugins are designed to work together seamlessly without conflicts.

How frequent are updates?

We release updates regularly based on user feedback and Minecraft version updates to ensure compatibility and new features.
